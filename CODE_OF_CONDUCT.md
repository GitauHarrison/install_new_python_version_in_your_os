# Code of Conduct for the Python Version & pyenv Helper

This project aims to make Python environment setup easier and more reliable. We want contributions and discussions to be constructive and welcoming.

## Expectations

We expect all participants (maintainers, contributors, users) to:

- Be respectful and considerate in all interactions.
- Assume good intentions and seek clarification before reacting.
- Provide specific, actionable feedback when pointing out issues.
- Use inclusive language and be mindful of different levels of experience.

## Unacceptable behavior

The following behaviors are not acceptable in this project:

- Harassment, discrimination, or derogatory comments.
- Threats of violence or encouragement of self-harm.
- Repeated, unwanted contact or attention.
- Publishing others' private information without consent.
- Any conduct that would be inappropriate in a professional setting.

## Scope

This Code of Conduct applies to all interactions within this repository (issues, pull requests, reviews, comments) and to any other communication channels explicitly associated with this project.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, please contact the project maintainer using the contact information provided in the repository (for example, in the README or project metadata).

When reporting, include as much relevant detail as possible:

- What happened.
- Where it happened (issue, PR, commit, etc.).
- When it happened.
- Any additional context that may help understand the situation.

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct. They may take any action they deem appropriate, including but not limited to:

- A private warning.
- Temporary or permanent restriction from contributing to this project.

By participating in this project, you agree to follow this Code of Conduct and help maintain a supportive, respectful environment for everyone.
